## Directional atomic layer etching of MgO-doped lithium niobate using Br-based plasma
- **URL**: http://arxiv.org/abs/2511.01825v2
- **AI解析**: 
ニオブ酸リチウム (LiNbO$_3$, LN) は、光通信から量子情報処理に至る幅広い集積フォトニクス応用で注目される非線形光学材料です。しかし、薄膜ニオブ酸リチウム (TFLN) デバイスの性能は、サイドウォール表面粗さやチップ上での形状不均一性といった作製上の不完全性によって現在制限されており、これらの課題克服には原子層エッチング (ALE) が有望視されています。

LNに対する等方性ALEプロセスは既に報告されているものの、パターン転写のための方向性ALEプロセスの欠如や、不揮発性化合物の再付着といった課題があり、LN作製全体をALEで完遂するには至っていません。

本論文では、MgOドープニオブ酸リチウムに対して、表面改質用のHBr/BCl$_3$/Arプラズマと除去用のArプラズマを連続的に曝露する**方向性ALEプロセス**を報告しています。HBrケミストリーは、F系およびCl系プラズマと比較して再付着を低減することが確認され、これはBr系生成物の蒸気圧が高いことに起因すると考えられます。

このプロセスによりエッチングされた回折格子パターン（合計エッチング深さ220 nm）は、最小150 nmのギャップにおいても**アスペクト比依存性エッチング (ARDE)** を示しませんでした。これは、同じエッチング深さで300 nmのギャップでもARDEが顕著に現れるイオンミリングとは対照的であり、本プロセスの優位性を示しています。

さらに、HBrプラズマケミストリーは、H$_2$プラズマとHBr/BCl$_3$/Arプラズマを連続的に曝露する**等方性ALEプロセス**もサポートすることが見出されました。

これらの方向性ALEおよび等方性ALEプロセスを組み合わせることで、TFLNデバイスの完全な作製プロセスを実現し、既存のイオンミリングに起因する不完全性を排除できる可能性が示唆されます。

## A transmon qubit realized by exploiting the superconductor-insulator transition
- **URL**: http://arxiv.org/abs/2510.19983v1
- **AI解析**: 
この論文は、実用的な量子コンピュータ実現に向けて有望なプラットフォームである超伝導量子ビットにおいて、スケーラビリティを阻害する既存のジョセフソン接合の課題を克服する新しいアプローチを提案しています。

**既存ジョセフソン接合の課題点:**
1.  **極低温動作の必要性:** 一般的なアルミニウム製接合では、超伝導ギャップが小さいため、ミリケルビン級の極低温環境が不可欠。
2.  **散逸の発生:** 異なる材料の界面が存在することで、エネルギー散逸が生じる。
3.  **不必要な静電容量:** サンドイッチ構造が高周波アプリケーションにおいて、望ましくない静電容量を付加する。

**本研究の提案と実現:**
本研究では、これら三つの課題全てに対処するため、超伝導-絶縁体転移（SIT）を応用した新規な超伝導弱結合を開発しました。具体的には、単一層の窒化ニオブ（NbN）薄膜を、原子層堆積法（ALD）と原子層エッチング法（ALE）のみを用いて局所的に薄膜化します。これにより、膜厚に駆動されるSIT特性を巧みに利用し、弱結合を形成しています。

この新しい弱結合を用いて、「planaron」と名付けられたトランスモン型量子ビットが作製されました。その測定結果は、非調和性 $\alpha/2\pi = 235$ MHz、現在の線幅 $\kappa/2\pi = 15$ MHzを示しています。

**優位性と将来性:**
*   **高温動作の可能性:** 窒化ニオブの高い超伝導ギャップは、将来のデバイスにおいて、より高い温度での動作を可能にする可能性を秘めています。
*   **損失と容量の削減:** 弱結合の完全に平面的な構造は、余分な材料界面とそれに伴う散逸、および不必要な静電容量を排除します。
*   **基礎研究への貢献:** SIT近傍の微小な材料領域を調査することは、散逸やサイズ効果を含むSITの根本的な性質に関する新たな知見をもたらすことが期待されます。

結論として、本研究は、既存の課題を克服し、よりスケーラブルで高性能な超伝導量子ビットの実現に向けた画期的な技術的進展を示すものです。

## Etching-to-deposition transition in SiO$_2$/Si$_3$N$_4$ using CH$_x$F$_y$ ion-based plasma etching: An atomistic study with neural network potentials
- **URL**: http://arxiv.org/abs/2508.00327v1
- **AI解析**: 
## 半導体論文要約：CHxFyイオンベースプラズマエッチングにおけるSiO2/Si3N4のエッチングから堆積への転移：ニューラルネットワークポテンシャルを用いた原子レベル研究

### 概要

半導体製造において不可欠なプラズマエッチングプロセスでは、ハイドロフルオロカーボンがエッチング剤および炭素膜形成の前駆体として利用されます。このプロセスにおいて、高いSiO2/Si3N4選択性や原子層エッチング (ALE) を実現するためには、膜成長の精密な制御が不可欠です。

本研究は、CHxFyイオンを用いたプラズマエッチングにおけるSiO2およびSi3N4表面の挙動を原子レベルで解明するため、**ニューラルネットワークポテンシャル (NNP)** を開発しました。

### 主要な研究手法と成果

1.  **NNPの開発と効率的なサンプリング手法:**
    *   多様な局所構造を効率的にサンプリングするため、イオン-基板の組み合わせを網羅的に列挙することなく、**高温・低密度の分子動力学 (MD) シミュレーションとベースライン参照構造を組み合わせた「蒸気-表面サンプリング」手法**を提案しました。
    *   繰り返し学習によって最適化されたNNPは、MDシミュレーションにおいて実験結果とよく一致するエッチング特性を示しました。

2.  **炭素層形成メカニズムの解明:**
    *   詳細な解析の結果、SiO2とSi3N4における炭素層形成メカニズムに明確な違いがあることが明らかになりました。
    *   **SiO2表面**では、炭素-酸素副生成物の**揮発性が高い**ため、炭素層の堆積が抑制され、エッチングが進行しやすい傾向にあります。
    *   対照的に、**Si3N4表面**では、揮発性炭素-窒素種の生成が**抑制される**ため、炭素層が形成されやすく、これがエッチングから堆積への転換を促す主要因となります。

### 結論と展望

この計算フレームワークは、プラズマ暴露下における原子レベルの表面改質を定量的に予測することを可能にします。これにより、半導体製造プロセスにおける詳細なメカニズムへの深い洞察をもたらし、将来的にはマルチスケールプロセスモデリングへの統合の基盤となることが期待されます。

## Atomic layer etching of InGaAs using sequential exposures of atomic hydrogen and oxygen gas
- **URL**: http://arxiv.org/abs/2506.19749v1
- **AI解析**: 
本論文は、InP HEMTなどのIII-V族半導体デバイスにおいて、サブサーフェスエッチング損傷やウェハ面内でのエッチング深さの不均一性が高性能化と高歩留まりを阻害するという課題に着目しています。

この課題を克服するため、著者らは原子層エッチング（ALE）技術が持つオングストロームスケールの高精度なエッチング、低ダメージ性、および本質的なウェハ全面均一性という利点に注目しました。

本研究では、InGaAsに対するALEプロセスが報告されています。このプロセスは、原子状水素と酸素ガスを交互に順次暴露することで行われます。その結果、350 °Cにおいて0.095 Å/cycleというエッチングレートがex-situ分光エリプソメトリーによって確認されました。特筆すべきは、200サイクルものALE処理後もサンプルの表面が原子レベルで平滑性を維持していた点です。

このALEプロセスは、InP HEMTの製造におけるゲートリセスエッチング工程に応用することで、デバイスのマイクロ波性能と歩留まりの大幅な改善に貢献する可能性を秘めていると結論付けられています。
